Relevant Courses

Architecture, Systems, Concurrency and Energy in Computation (ECE 3058)
This course taught me about cache simulation, virtual memory management, single and multi-level processors, and the intricate interplay between hardware and software. By simulating hardware tools using software, such as C, RISC-V, and Verilog, we gained practical insights into microarchitectural choices and the impact of design decisions. Additionally, we delved into hardware description languages (HDLs) like Verilog and System Verilog, honing our ability to enhance the efficiency of computer systems. This course equipped us with the skills to be effective computer architects with an understanding of computing systems.

Circuit Analysis (ECE 2040)
In ECE 2040, we explored circuit analysis and design, focusing on RLC circuits. We applied principles of engineering, science, and mathematics to solve complex problems using techniques like node and loop analysis. We also studied linearity, steady-state behavior, AC circuits, frequency response, and the Laplace transform. This course equipped us with the skills to analyze circuits, conduct experiments, and draw conclusions.

Data Structures and Algorithms (CS 1332)
In this course we focused on enhancing our Java programming skills and gaining proficiency in common data structures used in software development. We implemented low-level versions of data structures such as arrays, ArrayLists, LinkedLists (Singular, Doubly, Circular), Stacks, Queues, Deques, Priority Queues, various trees (Binary, Binary Search, AVL, Splay, Heaps, 2-4 Trees), Hash Maps/Tables, and graphs. We also learned and applied algorithms for sorting, pattern matching, graph traversal, and dynamic programming. Understanding Big-O notation helped us make informed decisions on data structure and algorithm selection for programming problems. Additionally, we improved our ability to test and debug programs effectively.

Digital Design Lab (ECE 2031)
We gained hands-on experience in designing, building, and testing digital circuits using programmable logic devices. Through labo projects, we worked with schematic capture, logic simulation, and HDL-based logic synthesis on FPGAs. We explored topics such as combinational logic circuits, sequential circuit elements, state machines, and basic computer design. Additionally, we learned to write machine language and assembly language programs and implemented them on FPGA boards. The course fostered team-building skills through design projects and emphasized technical writing skills through lab reports and project documentation.

Digital System Design (ECE 2020)
Throughout this course, we explored various levels of abstraction in digital computing systems, starting from electronic switches and DeMorgan equivalent gates and progressing to programmable machines. We gained hands-on experience in designing and simulating Boolean expressions, implementing encoders, decoders, and multiplexers, and analyzing combinatorial logic delay and energy dissipation. We learned about binary representations of different value types and performed arithmetic operations with signed and unsigned values. Additionally, we studied sequential logic, state machines, memory systems, and datapaths. The course provided us with the knowledge and skills to develop, analyze, and optimize digital designs at different layers of a computing platform.

Discrete Math (CS 2050)
This course focused on key concepts such as propositions, proofs, propositional logic, induction, recursion, counting, probability, number theory, and models of computation. The course aimed to help us develop logical reasoning skills, write formal proofs, understand encryption techniques, utilize regular expressions, and design computational models.

Feedback Control Systems (ECE 3550)
Throughout this course, we explored concepts such as feedback control, Laplace transforms, system response, stability analysis, and control system design. We applied graphical analysis techniques like root locus plots and Nyquist plots to analyze and design control systems effectively. Our course objectives included developing the ability to identify and solve complex engineering problems, applying engineering design principles, conducting experiments, and acquiring new knowledge. We covered topics such as signal operations, linear time-invariant ordinary differential equations, transfer functions, feedback control, and stability analysis. By the end of the course, we had gained a strong foundation in control engineering principles and techniques.

Intro to Object Oriented Programming (CS 1331)
This course introduced us to Object Oriented Programming. We learned object-oriented programming principles and applied them to construct Java programs. Our objectives were to write medium-sized Java programs, use appropriate algorithms and data structures, and understand event-driven programming for GUI programs. By the end, we had gained proficiency in object-oriented programming, improved our Java skills, and developed the ability to create effective GUI programs.

Intro to Signal Processing (ECE 2026)
Here, we focused on signal processing and its mathematical representation. We learned to write Matlab code for signal processing systems and analyze signals in terms of their frequency content. Understanding system behavior and analyzing linear systems using Fourier transform and frequency response were also key components. We delved into mixed analog-digital systems, sampling operations, and digital filters. Additionally, we utilized the z-transform to analyze discrete-time systems and applied complex exponential notation to describe signals and systems. We explored various applications of signal processing, such as audio and digital image processing.

Programming for Hardware/Software Systems (ECE 2035)
Here, we learned about the architecture and implementation of data structures like heaps and hash tables, and how heaps work in memory. Through MIPS and C programming languages, we developed these data structures from scratch, understanding their binary tree structure, heap property, hash functions, collision resolution strategies, and memory management. This hands-on experience deepened our understanding of these data structures and their interaction with computer architecture and hardware, enhancing our programming skills in MIPS and C.
